Podmień te pliki w głównym folderze projektu:
- types/pg.d.ts
- .gitignore

Po podmianie uruchom:
1. git add .
2. git commit -m "Fix pg TypeScript declarations"
3. git push

Jeśli w repo masz już wrzucone node_modules, samo dodanie .gitignore nie usunie ich z historii.
Na razie to nie blokuje deployu. Po udanym buildzie można zrobić porządek osobno.
